# Planer Produktywności

Wieloplatformowy system zarządzania zadaniami.

## Aplikacje
- **Web** — Angular
- **Desktop** — C# WinForms/WPF
- **Mobile** — C# .NET MAUI
- **Backend** — ASP.NET Core Web API + SQLite

## Jak uruchomić

### Backend
1. Wejdź do folderu `backend`
2. Uruchom `dotnet run`
3. API działa na `http://localhost:5000`

### Aplikacja webowa
1. Zainstaluj Node.js
2. Wejdź do folderu `web-app`
3. Uruchom `npm install`, potem `ng serve`
